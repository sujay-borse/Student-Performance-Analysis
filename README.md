# Student Performance Analysis System

## Project Description

This project analyzes student marks using Python, Pandas, NumPy, and
Matplotlib.

## Features

-   Calculate total and average marks
-   Find highest and lowest average
-   Calculate pass percentage
-   Generate bar chart and pie chart

## How to Run

1.  Install dependencies: pip install -r requirements.txt

2.  Run the project: python main.py

Charts will be generated in the project folder.
